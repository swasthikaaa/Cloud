DigiWorks Studio Cloud Migration

Transforming on-premises infrastructure into scalable, secure AWS cloud architecture

Project Overview

This project involved migrating DigiWorks Studio’s on-premises infrastructure to AWS using CloudFormation automation, delivering scalability, cost efficiency, and security improvements.

Key Skills & Highlights

Click a skill to view details.

Migration Strategy

  • Web Servers → EC2 with Auto Scaling & Load Balancer
  • Database → RDS MySQL with Multi-AZ & automated backups
  • File Storage → S3 with lifecycle rules to Glacier
  • Email → Microsoft 365 SaaS migration
  • VPN → Site-to-Site hybrid connectivity

Key Benefits

  • Cost Optimization through Auto Scaling and S3 lifecycle policies
  • High Reliability with Multi-AZ RDS & CloudFormation automation
  • Security & Compliance via GuardDuty, CloudTrail, IAM policies
  • Operational Agility: rapid deployment of environments via IaC
